Electronic Chart Work — CHANGELOG

Version: v1-baseline
Date: 2026-03-17

Summary
- Established a baseline bundle and agreed workflow for future changes.
- This bundle is the canonical current version to build on from here.

Current feature set
- Pan tool
- FIX point tool
- WPT point tool
- CTS line tool
- CMG line tool
- Set line tool
- P-Line tool
- Freehand scribble tool
- Select tool
- Delete tool
- Save / Load / Clear
- Option B editing action bar

Behaviour included
- Point and line tools revert to Pan after use
- Freehand remains active until another tool is selected
- Select toggles off when the selected artefact is tapped again
- FIX uses click/drag/release placement with temporary coordinate box
- WPT uses click/drag/release placement with temporary coordinate box
- Temporary coordinate box appears top-right of the point during placement
- Line tools use:
  1) first tap to set start point
  2) second press-and-hold to define temporary end point
  3) release to commit final endpoint
- Lines can be started from an existing point symbol
- P-Line chevron is at the start of the line
- Permanent line labels are shown:
  - direction above the line in format (045)T
  - distance below the line in format 10.2nm
- Permanent line labels never render upside down
- Line selection uses an enlarged hit area
- Freehand restored in current baseline

Editing / object management
- Selecting an artefact opens action bar with:
  - Move
  - Edit
  - Delete
  - Done
- Points can be moved/edited
- Lines can be moved/edited
- Freehand can be moved
- Toolbar Delete remains available

Notes on development workflow
- Future changes should always be made against this baseline bundle.
- New releases should be saved as versioned zips.
- Regressions should be checked against REGRESSION_CHECKLIST.txt.
